--- TIP 2 ---
WeClaw skills = markdown files. No Python changes. Edit `skills/wechat_removal.md` to tune detection rules for your group type.

Extensible by design.

#WeClaw #OpenSource

--- TIP 3 ---
Dual-model setup in WeClaw:
• Heavy model: coordinate prediction, message parsing
• Fast model: yes/no verification

Cuts latency + cost. Config in model.yaml.

#AIAutomation #LLM

--- TIP 4 ---
WeClaw's Find-Click-Verify pattern:
1. Vision query returns coords
2. Click
3. Fast model verifies success

Same pattern scales to any desktop app. CUA makes it reusable.

#WeClaw #ComputerUseAgents
