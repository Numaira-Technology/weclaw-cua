Show HN: WeClaw – AI agent that scans WeChat groups for scams (open source)

--- FIRST COMMENT (post immediately after submission — this becomes the "body" of your post) ---
We built WeClaw to solve a problem we had: moderating WeChat groups manually was eating hours every week. Scammers and spammers were slipping through.

WeClaw is an AI agent that runs on your desktop, uses vision (screenshots + LLM) to read the WeChat UI, and detects suspicious patterns—phishing links, impersonation, fake accounts. It proposes removals; you confirm. No auto-removal. Human-in-the-loop by design.

Technical details:
- Built on CUA (Computer Use Agents) — vision + control, no reverse-engineered APIs
- Find-Click-Verify pattern with dual-model routing (heavy for coordinates, fast for verification)
- Skills = markdown files, so behavior is tunable without code changes
- Windows + macOS, OpenRouter for LLMs (Claude, GPT-4o, Gemini, Qwen)
- 100% open source, MIT

Roadmap: reply generation, friend request verification, more automation. We're building what the community asks for.
