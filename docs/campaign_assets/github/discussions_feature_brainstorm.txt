WeClaw roadmap brainstorm — share your ideas

--- BODY ---
We're planning the next phase of WeClaw and want your input.

**Current capabilities:**
- Scam/fraud detection in WeChat groups
- Human-in-the-loop removal
- Skills system for customization
- Windows + macOS support

**Ideas we're considering:**
1. **Reply generation** — Draft or suggest replies to group messages
2. **Friend request verification** — Screen incoming requests before accepting
3. **Batch operations** — Handle multiple groups in one run
4. **Custom skill templates** — Community-contributed playbooks (business vs. family vs. community)
5. **OpenClaw deep integration** — Seamless command chaining

**Your turn:**
- What's missing?
- What would you use daily?
- What would make you recommend WeClaw to others?

Drop a comment. We read everything.
